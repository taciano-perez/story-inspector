Source Code

Story Inspector is Free Open Source Software (FOSS) published under the GPLv2 license.

The source code is available at:

https://github.com/taciano-perez/story-inspector  

Story Inspector is based on the following software packages. We thank the authors of said packages for making them available as free software.

  • Stanford CoreNLP - https://stanfordnlp.github.io/CoreNLP/
  • NRC Word-Emotion Association Lexicon (EmoLex) - https://saifmohammad.com/WebPages/NRC-Emotion-Lexicon.htm
  • Spring Boot - https://spring.io/projects/spring-boot
  • Java FX - https://openjfx.io/
